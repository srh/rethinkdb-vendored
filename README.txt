                          rethinkdb-vendored
                          ==================

This repo was used during the integration of Duktape and then QuickJS
JavaScript engines into RethinkDB.  It checked out certain commits in
RethinkDB's history.

This repository is kept around so that the abandoned Duktape branches
can still be built and run as-is.

(The QuickJS branch has been merged into v2.4.x and no longer uses the
vendored repo -- it uses RethinkDB's package system.)
